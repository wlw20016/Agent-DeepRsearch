# 场景
working1 : 我在看项目，我问了AI关于项目中的一个问题
working2 ：Vibe Coding一个想法
working3 ：问了另一个AI关于某些技术栈的知识
（边学习边让agent实现自己的想法，不耽误自己的学习时间）

# 痛点
顾不过来，看完working1可能就忘记看working2和3；
注意力不集中了就。

# 方案
Agent工作统一管理方案：
桌面上开一个小应用，可以配置好正在工作的agent，实时跟踪agent状态，agent工作结束以后，状态就变为已结束，正在工作中状态就对应工作中。需要人类授权就标记为待授权，然后后面给出选项，点击即可授权。点击相应的agent就能立刻打开相应的应用。
